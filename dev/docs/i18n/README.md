# UI translations (i18n)

UI copy only. All other project text is English — see `../LOCALE.md`.

| File | Locale | Role |
|------|--------|------|
| `en.json` | `en` | Default and fallback |
| `pt-BR.json` | `pt-BR` | Portuguese (Brazil) |

Add new locales by copying `en.json` to `<tag>.json` and translating values (keys stay the same).
